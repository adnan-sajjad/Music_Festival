# Music Festival Revenue & Artist Performance Analysis

End-to-end data analysis project on Indian music festival data — data cleaning, exploratory analysis, and a Power BI dashboard. Built with **MySQL**, **Python**, and **Power BI**.

## Project Overview

This project analyzes 105 festival events across Indian cities to answer:
- Which festivals, cities, and artists generate the most revenue?
- How does ticket pricing vary by artist?
- What does the monthly revenue trend look like, and where are the growth/decline points?
- How does an artist's streaming popularity relate to their touring revenue?

The same cleaning and analysis logic is implemented twice — once in SQL, once in Python — so the repo demonstrates both skill sets on the same dataset.

## Data

Two raw datasets:

| File | Rows | Description |
|---|---|---|
| `musicfestival.csv` | 105 | Festival name, city, date, headliner, ticket price, attendance, revenue breakdown |
| `Artist_Popularity_Data.csv` | 10 | Artist genre, average ticket price, social/streaming metrics, tour impact |

**Known data quality issues (handled in cleaning):**
- Dates arrive in two different formats (`dd/mm/yyyy` and `dd-mm-yyyy`)
- `Tour_Name` is missing for ~60% of rows → labeled `"Unknown Tour"`
- `Ticket_Price` and `Total_Attendance` have ~10 missing values each → imputed with the column mean
- One artist record (Bryan Adams) is missing average attendance → backfilled from festival-level data

## How to Run the Python Analysis

```bash
pip install -r requirements.txt
python python/analysis.py
```

This will:
1. Load and clean both raw CSVs (mirrors the SQL cleaning steps)
2. Print EDA and business-question results to the console
3. Save 5 chart images to `images/`
4. Save cleaned CSVs to `data/cleaned/`

## Key Findings

- **Ziro Festival Arunachal** is the top revenue-generating festival, contributing ~16% of total revenue across all events.
- Revenue is concentrated: the top 5 festivals account for roughly two-thirds of total revenue.
- **DJ Snake** and **Bryan Adams** command the highest average ticket prices, but that doesn't translate directly into total revenue — audience size and city matter more.
- Monthly revenue is volatile (month-over-month swings from -87% to +430%), which reflects **small sample size per month** rather than a genuine seasonal pattern — worth noting if this dataset is used to justify scheduling decisions.
- Genre popularity in the artist dataset is dominated by Bollywood (3 of 10 artists), so genre-level conclusions from that file should be treated as directional, not statistically robust.
